# task-manager-api

## Run locally
```
> npm run dev
```

## Heroku Commands

1. Create Heroku App
```
> heroku create {app-name}
```

2. Set and remove Heroku environment variables
```
> heroku config:set key=value

> heroku config:unset key=value
```

3. Display heroku environment variables
```
> heroku config
```

4. deploy to Heroku
```
> git push heroku master
```
